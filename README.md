# 🧠 Flask Quiz Web Application

Простое и стильное веб-приложение для прохождения викторин, написанное на Python с использованием микрофреймворка Flask и базы данных SQLite3.

## ✨ Особенности проекта
* **Выбор тем**: Динамическая загрузка доступных викторин из базы данных.
* **Умный счетчик**: Система сессий (`flask.session`) считает общее количество вопросов и правильных ответов пользователя.
* **Случайный порядок**: Варианты ответов перемешиваются при каждом запуске вопроса.
* **Адаптивный интерфейс**: Современный центрированный дизайн с кастомным фоном и CSS-стилизацией ответов в виде сетки.
* **Валидация**: Защита от пропуска вопросов (перейти к следующему можно только выбрав ответ).

## 📂 Структура проекта
```text
твой_проект/
│
├── app.py               # Основной файл Flask-приложения
├── db_init.py           # Скрипт создания и заполнения базы данных
├── quiz.sqlite          # Файл базы данных SQLite3 (генерируется скриптом)
└── static/              # Папка со статическими файлами сайта

```

## 🚀 Быстрый запуск

### 1. Подготовка базы данных
Перед первым запуском сайта необходимо создать и наполнить базу данных вопросами. Для этого запустите скрипт инициализации:
```bash
python db_init.py
```
*После этого в корне появится файл `quiz.sqlite`.*

### 2. Запуск веб-сервера
Запустите основной скрипт приложения:
```bash
python app.py
```

### 3. Открытие в браузере
Перейдите по адресу, который отобразится в терминале (по умолчанию):
```text
http://127.0.0
```

## 🛠️ Стек технологий

* **Frontend**: HTML5, CSS3 (CSS Grid, Flexbox)
